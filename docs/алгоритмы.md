# Использованные алгоритмы и структуры.
## Простые
* AABB / Circle Overlap (коллизии)
* Camera Offset
* Observer / Event pattern с приоритетами для EventBus.
* GameLoop + delta_time

* Random Walk (для пути)
## Средние
* UniformGrid для областей коллизий и прицеливания.
* Wave Function Collapse (для окружения)
