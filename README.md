# Not yours shelter!

Название: **Not yours shelter!**

Жанр: Tower Defence с элементами Survival, Sandbox и Roguelike.

Идея в одном предложении: "Тавер-дефенс, где ты путешествуешь по этажаме, на которых тебе нужно построить убежище для защиты от волн врагов!"

Основная фишка: Свобода постройки, персонаж с экипировкой, турели, генерация уровня на этаже.
